# GlistApp

App project template for [GlistEngine](https://github.com/GlistEngine/GlistEngine)

Clone this repository under `glist/myglistapps` in your GlistEngine installation directory to initialize a new GlistEngine game project. More information can be found on the [GlistEngine readme](https://github.com/GlistEngine/GlistEngine/blob/main/README.md).
